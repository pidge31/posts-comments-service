# posts-comments-service

GraphQL-сервис для управления постами и комментариями с поддержкой подписок в реальном времени.

## Возможности

- Создание, чтение и удаление постов
- Вложенные комментарии (любая глубина вложенности)
- Управление комментариями к посту (включить/выключить)
- Пагинация на основе курсора (Relay-style)
- Подписки на новые комментарии через WebSocket
- Два варианта хранилища: in-memory и PostgreSQL
- Мягкое удаление комментариев

## Технологии

- Go 1.25
- [gqlgen](https://github.com/99designs/gqlgen) — кодогенерация GraphQL
- PostgreSQL 16 (опционально)
- Docker / Docker Compose

## Быстрый старт

### С Docker Compose (PostgreSQL)

```bash
docker compose up --build
```

Сервис запустится на `http://localhost:8080/query`.  
GraphQL Playground доступен по адресу `http://localhost:8080/`.

### Локально (in-memory хранилище)

```bash
cp .env.example .env
go run ./cmd/api
```

## Конфигурация

Настройки задаются через переменные окружения:

| Переменная     | По умолчанию | Описание                                |
|----------------|--------------|-----------------------------------------|
| `PORT`         | `8080`       | Порт HTTP-сервера                       |
| `STORAGE_TYPE` | `memory`     | Тип хранилища: `memory` или `postgres`  |
| `DATABASE_URL` | —            | Строка подключения к PostgreSQL         |

Пример `.env`:

```env
PORT=8080
STORAGE_TYPE=memory
DATABASE_URL=postgres://postgres:postgres@localhost:5433/posts_comments?sslmode=disable
```

## GraphQL API

### Mutation

| Mutation                 | Описание                                      |
|--------------------------|-----------------------------------------------|
| `createPost`             | Создать пост                                  |
| `deletePost`             | Удалить пост (только автор)                   |
| `setPostCommentsEnabled` | Включить/выключить комментарии (только автор) |
| `addComment`             | Добавить комментарий или ответ                |
| `deleteComment`          | Мягкое удаление комментария (только автор)    |

### Запросы

| Запрос     | Описание                                        |
|------------|-------------------------------------------------|
| `posts`    | Список постов с пагинацией                      |
| `post`     | Пост с комментариями первого уровня             |
| `comment`  | Комментарий с ответами (любой уровень вложения) |

###  Subscription

|  Subscription  | Описание                                     |
|----------------|----------------------------------------------|
| `commentAdded` | Новые комментарии к посту в реальном времени |

### Ограничения

| Поле              | Максимум         |
|-------------------|------------------|
| Заголовок поста   | 200 символов     |
| Тело поста        | 10 000 символов  |
| Текст комментария | 2 000 символов   |

## Команды

```bash
make run               # Запустить сервис локально
make test              # Юнит-тесты
make test-race         # Тесты с детектором гонок
make test-integration  # Интеграционные тесты (требуется PostgreSQL)
make generate          # Перегенерировать GraphQL-код
make docker-up         # Запустить через Docker Compose
make docker-down       # Остановить и удалить контейнеры и тома
make migrate-up        # Применить миграции
make migrate-down      # Откатить миграции
```
